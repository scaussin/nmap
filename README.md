# nmap
nmap - Network exploration tool and security / port scanner

# Usage
```
nmap <adresse> <port_to_start> <n_ports_to_Scan>
```

# Example:
```
$> sudo nmap 192.168.1.1 0 200
PORT    STATE
53      OPEN
80      OPEN
113     CLOSE
135     CLOSE
137     CLOSE
138     CLOSE
139     OPEN
193 filtered ports
```
